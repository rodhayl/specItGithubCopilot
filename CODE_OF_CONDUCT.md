# Code of Conduct

## Our Commitment

This project is committed to a respectful, harassment-free experience for everyone, regardless of background, identity, or experience level.

## Expected Behavior

- Be respectful and constructive in discussions and reviews.
- Focus feedback on code, design, and behavior, not people.
- Assume good intent and ask clarifying questions before escalating.
- Help keep collaboration accessible for new contributors.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, intimidation, or inflammatory language
- Publishing private information without consent
- Repeatedly derailing technical discussions

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct and may remove comments, issues, or contributions that violate it.

## Reporting

Report conduct concerns privately to the maintainers through GitHub profile contact channels.

All reports will be reviewed promptly and handled with discretion.
